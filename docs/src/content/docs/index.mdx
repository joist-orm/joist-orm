## Why Joist?

Joist is a TypeScript ORM that focuses on helping you build rich domain models.

Unlike most other ORMs, Joist gives you a great developer experience, with rich autocompletion, clean APIs, and a simple set of tools for modeling your domain, all while avoiding N+1 performance problems.

Learn more in the [Why Joist?](/why-joist/) section.
